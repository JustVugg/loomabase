How to prove it

        
    
          
  1. Reset the demo server and both devices.
    2. 
          
  2. On the desktop, create the todo and sync.
    3. 
          
  3. On the phone, sync once to pull the todo.
    4. 
          
  4. Enable offline mode on both devices.
    5. 
          
  5. On the phone, edit the title. On the desktop, toggle completed.
    6. 
          
  6. Disable offline mode. Sync phone, sync desktop, then sync phone again.
    7. 
        

        

          The final state should contain the phone title and the desktop
          completed flag. This works because Loomabase merges each column as a
          separate CRDT register ordered by Lamport clock and device ID.
